## Libraries

### Plots

Plots are generated using [`plotly`](https://plotly.com/python/line-charts/) library.

![alt text](image-1.png)

### Hashing 

Hashing is done using [`hashlib`](https://docs.python.org/3/library/hashlib.html) library which is a part of the Python standard library.

## Example for checksum

Checksum for [Kali Linux](https://www.kali.org/get-kali/#kali-virtual-machines) downloaded from the official website.

### sum on the website
![alt](image.png)

### sum with powershell build-in tools
```powershell
> Get-FileHash -Algorithm SHA256 ..\kali-linux-2024.1-virtualbox-amd64.7z

Algorithm       Hash                                                            
---------       ----                                                            
SHA256          80BA5FDD6037F9261A5F49A7F18A0ABB231551A1AEC2D97DA2344C2A0E72CED8
```

### checksum with python
using `hash_file()` function from `main.py` file.

```powershell
> python .\main.py

Hashing file: ./../kali-linux-2024.1-virtualbox-amd64.7z
Hash: 80ba5fdd6037f9261a5f49a7f18a0abb231551a1aec2d97da2344c2a0e72ced8
```
